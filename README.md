# studentprojectstorage
Student Project Storage System

A web-based inventory management system designed to track and manage project-related resources in a lab environment. The system allows users to monitor item availability, issue items to students, and manage resource usage efficiently.

Overview
---


This project is built to simplify how lab equipment and components are handled across multiple student teams. It provides a structured way to track whether items are:

Available
Currently in use
Borrowed by another team

The system ensures transparency and reduces conflicts in resource allocation.

Features
---

User authentication (admin and users)
Item tracking and status management
Issue and return system for lab resources
Real-time availability updates
Structured database for efficient queries

Tech Stack
---
Frontend: HTML, CSS, JavaScript
Backend: Python (Flask)
Database: MySQL

Purpose
---
This project is developed as part of a Database Systems Lab to demonstrate practical implementation of database design, backend integration, and frontend interaction in a real-world use case.
